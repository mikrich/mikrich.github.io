Remembering The Basics

  • <!-- bad, no focus -->
<a ng-click="doSomething()">Sample</a>
    <!-- good, can focus -->
<a href="#" ng-click="doSomething()">Sample</a>
<a tabindex="0" ng-click="doSomething()">Sample</a>
    <!-- better, focus for free -->
<button ng-click="doSomething()">Sample</button>
  • Evaluate plugins before you implement them

Managing Focus

  • Some native behavoir is lost
  • Focus is inconsistent between views
  • Need for greater control
<!-- Focus When Directive -->
<h1 focus-when="loaded === true">Heading</h1>

Handing Alerts

  • Dynamic alerts, validation etc.
  • Directives such as ng-if & ng-switch need consideration
  • Use live regions
    • <!-- Live Regions -->
<div aria-live="assertive">
  <div class="well" ng-if="main.alertAriaA === true">
    <p>This is an Alert with a live region - Assertive</p>
  </div>
</div>
demo
  • Test and review regularly
  • Ensure ideas are aligned
  • Include from the start, don't retrofit!
